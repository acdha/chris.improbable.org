CSS :fullscreen selector test

This document contains two elements which are selected by the :fullscreen pseudoclass. The first element uses separate CSS rules and should always display once the browser is in fullscreen mode. The second element combines the selectors into a single group and will never match.

Issues

  • Mozilla: https://bugzilla.mozilla.org/show_bug.cgi?id=1026762
  • Chrome: https://code.google.com/p/chromium/issues/detail?id=385842
  • WebKit: https://bugs.webkit.org/show_bug.cgi?id=133999

Separate Style Testcase 


            html:fullscreen #full-screen-only {
                color: green;
            }

            html:-moz-full-screen #full-screen-only {
                color: green;
            }

            html:-webkit-full-screen #full-screen-only {
                color: green;
            }

This element was matched by a single rule

Combined Style Testcase 


            html:fullscreen #full-screen-only-combo,
            html:-moz-full-screen #full-screen-only-combo,
            html:-webkit-full-screen #full-screen-only-combo {
                color: green;
            }

This element was selected by a list of selectors
