Fun with Eric Raymond

My resume has a polite note to headhunters who happen across it reminding them that I'm not looking for work. I included a comment that the page still existed because I dislike link-rot and included a link to the Jargon File, which has been maintained by Eric Raymond of OpenSource fame / notoriety.

An alert co-worker noticed that the link has rotted recently - any request to www.tuxedo.org seems to be redirected to a random OpenSource / gun-nut page.

I have replaced it with a link to useit.com. Hopefully Jakob Nielsen will be more reliable...
