Rendezvous then I'm through with you...

  • Rendezvous sounds nice. Except that it doesn't even check normal DNS so admins get stuck having to setup a multicast DNS responder. Oh, and there doesn't appear to be much on the mDNS server front other than specs and talk. In particular I have yet to find a message from anyone actually using it.
  • Carbon apps broke on my system. Ouch...
  • Someone at Apple removed the Command-Shift backwards cycle after Command-Tab.
  • If your system supports QuartzExtreme, try /System/Library/Frameworks/ScreenSaver.framework/Resources/ScreenSaverEngine.app/Contents/MacOS/ScreenSaverEngine -background & - it'll run your current screensaver as your desktop wall paper
