The Clubhouse launches on Microsoft.com

The name says all you need to know about the value of branding agencies and focus groups

Last week, Microsoft announced that it was expanding its Clubhouse and would be launching a new site to celebrate. That site is now live at clubhouse.microsoft.com. The biggest change is that the Clubhouse community will now encompass Windows, in addition to Windows Live, a transformation that reflects Microsoft recently folding its Windows Live team into the Windows team. It's also worth noting that these moves are happening simultaneously with the launch of Microsoft's new Rookies marketing campaign, which features children using Windows Live products on Windows.

Back in August 2008, Microsoft launched the Windows Live Community Clubhouse, a site dedicated to users who are interested in sharing their Windows Live tips and learning new things from users both inside and outside of Microsoft.
