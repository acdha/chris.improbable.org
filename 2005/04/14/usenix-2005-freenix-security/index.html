USENIX 2005: FREENIX Security

Ourmon

  • In production at PSU since 2001
  • Basic architecture: front end collector forward summaries out of band to analysis system
  • Significant BPF load - requires a tuned kernel, large buffers, etc.
  • Next release will focus on anomaly detection

  • The Bro Observatory

    • Companion to Bro IDS
    • Focus on analysis, not real-time alerts: filtering, aggregation, etc.

    NetState

    • “State of the Network” view - passive network monitoring for running services (including non-standard ports), operating system versions, and limited client version detection.
