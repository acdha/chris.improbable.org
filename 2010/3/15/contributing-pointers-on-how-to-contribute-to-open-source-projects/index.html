Contributing -- pointers on how to contribute to open source projects

Interesting project by Brad Fitz

This is Contributing, a community resource dedicated to keeping up-to-date information on how to contribute to open source projects.

It gives projects a stable URL which they can put in their documentation (and/or CONTRIBUTING files) providing an abstraction layer making sure even old downloads of projects still contain a pointer to recent pointers to how-to-contribute docs.

Also, this site allows for project take-overs, if the original maintainer(s) disappear and don't keep their information fresh.

Source: http://contributing.appspot.com/
